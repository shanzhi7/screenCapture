# 开发计划（同步版）

## Phase A 已完成（UI 与基础截图）

- 主界面重构（HyperOS 风格）
- 全屏截图 / 区域截图闭环
- 保存与复制能力
- 覆盖层工具条流程

- Windows 全局热键注册（`Ctrl+Shift+A`）

4. 配置命名品牌化与旧配置自动迁移（`SettingsService`）

## Phase E 进行中（稳定性强化）

1. 长截图复杂页面拼接鲁棒性继续优化
2. 混合 DPI / 多屏边界场景回归测试
