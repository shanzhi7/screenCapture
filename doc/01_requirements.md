# 功能需求文档

## 1 核心功能

### 1.1 区域截图

用户可以：

1. 点击截图按钮
2. 进入截图模式
3. 鼠标拖动选择区域
4. 释放鼠标完成截图

系统行为：

- 截取选定区域
- 生成 QPixmap

---

### 1.2 截图预览

截图完成后：

- 在主窗口显示预览
- 支持重新截图

---

### 1.3 保存图片

用户可以：

- 点击保存
- 选择路径
- 保存 PNG

---

### 1.4 复制到剪贴板

用户可以：

- 点击复制
- 图片进入系统剪贴板

---

## 2 交互需求

截图模式下：

| 操作  | 行为     |
| ----- | -------- |
| ESC   | 取消截图 |
| Enter | 确认截图 |
| 拖动  | 选择区域 |

---

## 3 非功能需求

### 性能

截图延迟 < 100ms

### 资源占用

内存 < 100MB

### 兼容

Windows 10+
