# 功能设计文档（按当前按钮与流程更新）

> 更新时间：2026-03-06

## 1 当前可用功能

### 1.1 全屏截图

入口：`modeFullButton` + `startCaptureButton`

流程：

1. 选择“全屏”模式
2. 点击“开始截图”
3. 调用 `QScreen::grabWindow(0)`（鼠标所在屏幕）
4. 更新右侧预览

### 1.2 区域截图

入口：`modeRegionButton` + `startCaptureButton`

流程：

1. 选择“区域”模式
2. 启动 `SelectionOverlay`
3. 鼠标拖拽选区
4. Enter 确认或工具条确认
5. 调用抓屏并更新预览

### 1.3 滚动截图（T101）

入口：`modeScrollButton` + `startCaptureButton`

流程：

1. 选择“滚动”模式
2. 启动 `SelectionOverlay` 选择滚动区域
3. 自动抓取首帧
4. 手动滚动目标内容并点击“追加一帧”
5. 点击“完成拼接”生成长图
6. 预览区显示拼接结果

说明：当前为最小可用版，采用手动推进滚动。

### 1.4 保存图片

入口：

- 主流程内（覆盖层工具条“保存”）
- 后续可挂接主界面按钮

行为：`QFileDialog` 选择路径后保存 PNG

### 1.5 提示反馈

- 使用 `ShowTip` 显示操作结果和占位提示

## 5 异常与边界处理

- 无屏幕可用：提示失败
- 选区过小：不执行截图
- 保存路径为空：取消保存流程
- 资源缺失：图标走文字兜底
- 滚动帧不足：提示无法拼接

---

## 6 性能目标（当前沿用）

- 截图延迟：< 100ms（单屏常规分辨率目标）
- UI 响应：< 16ms（常规交互）
- 内存占用：< 200MB（含历史缩略图缓存的阶段性目标）
