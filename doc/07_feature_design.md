# 功能设计文档（同步版）

> 更新时间：2026-03-06

## 1. 已实现功能

### 1.1 全屏截图

- 支持多屏环境选择目标显示器
- 抓图前临时隐藏主窗口，避免截入工具自身

### 1.2 区域截图

- 覆盖层拖拽选区
- 确认后自动复制到剪贴板
- 支持保存 PNG

### 1.3 长截图（区域模式内）

- 启动方式：区域选区 -> 工具条“长截图”
- 滚轮触发抓帧
- 使用 `LongCaptureStitcher` 增量拼接
- 结果支持复制与保存

### 1.4 提示系统

- 所有关键动作通过 `ShowTip` 提示

## 2. 部分实现功能

- 编辑工具按钮（画笔/矩形/圆形/马赛克/文字）仅 UI 入口，绘制逻辑未接入

## 3. 待实现功能

- 历史记录完整能力
- 设置持久化（热键、格式、自动保存）
- 编辑引擎与撤销重做

## 4. 风险与优化方向

- 长截图在复杂动态内容下仍可能偶发错位，需继续优化位移估算与异常帧过滤
- 多屏/DPI 混合场景需补充专项回归
