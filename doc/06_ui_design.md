# UI设计文档

## 1 UI总体设计

截图工具界面由三个部分组成：

1. 主窗口（MainWindow）
2. 截图遮罩层（SelectionOverlay）
3. 截图预览区域

UI设计原则：

- 简洁
- 快速操作
- 最少点击

---

# 2 主窗口设计

MainWindow 负责：

- 启动截图
- 显示截图
- 保存图片
- 复制图片

界面布局：

+----------------------------------+
 | ScreenCapture Tool               |
 +----------------------------------+
 |                                  |
 |        Screenshot Preview        |
 |                                  |
 |          (QLabel显示)             |
 |                                  |
 +----------------------------------+
 | [Capture] [Save] [Copy]          |
 +----------------------------------+

组件说明：

| 控件    | 类型        | 作用         |
| ------- | ----------- | ------------ |
| Capture | QPushButton | 开始截图     |
| Save    | QPushButton | 保存图片     |
| Copy    | QPushButton | 复制到剪贴板 |
| Preview | QLabel      | 显示截图     |

---

# 3 截图遮罩层

截图时会出现一个全屏透明窗口。

UI效果：

+----------------------------------+
 |                                  |
 |      半透明黑色遮罩               |
 |                                  |
 |        +-------------+           |
 |        |             |           |
 |        | 选区区域     |           |
 |        |             |           |
 |        +-------------+           |
 |                                  |
 +----------------------------------+

规则：

- 遮罩颜色：rgba(0,0,0,120)
- 选区区域保持原始亮度
- 选区边框颜色：蓝色

---

# 4 选区交互

鼠标行为：

| 操作     | 行为     |
| -------- | -------- |
| 鼠标按下 | 开始选区 |
| 鼠标拖动 | 更新选区 |
| 鼠标释放 | 完成选区 |

键盘行为：

| 按键  | 行为     |
| ----- | -------- |
| ESC   | 取消截图 |
| Enter | 确认截图 |

---

# 5 选区尺寸提示

截图时显示选区大小：

+--------------------+
 | 320 x 240          |
 +--------------------+

显示在选区右下角。

---

# 6 预览界面

截图完成后：

MainWindow显示截图。

+---------------------------+
 |                           |
 |        Screenshot         |
 |                           |
 +---------------------------+

预览图缩放适应窗口。
