# UI 设计文档（同步版）

> 更新时间：2026-03-07

## 1. 设计目标

- 简约、现代、轻量
- 主界面非透明
- 截图路径聚焦，关键反馈明确

## 2. 主界面结构

- 左侧导航栏（入口占位）
- 中央主内容区（快速截图 + 最近截图）
- 右侧信息区（预览格式 + 常用设置）

## 3. 快速截图区（当前）

- 模式按钮：`全屏`、`区域`
- `modeScrollButton` 保留控件但隐藏，不作为用户主路径
- `开始截图` 按钮触发当前模式流程

## 4. 覆盖层设计（SelectionOverlay）

- 半透明遮罩
- 选区轻微蓝色描边（仅交互态）
- 尺寸信息浮层
- 工具条：画笔/矩形/圆形/马赛克/文字/长截图/保存/取消/确认

## 5. 长截图 UI 行为

- 入口：区域选区后的“长截图”按钮
- 滚轮期间选区可视高度持续增长
- 主界面预览实时更新拼接结果
- 长截图状态提示仅作为交互反馈，不进入最终结果
- 选区贴近屏幕上下边缘时，主工具条与子工具条整体避让边界
- 完成后支持保存或复制

## 6. 关闭行为弹窗（CloseDialog）

- 使用独立 `.ui` 构建
- 配色与主界面一致（同一 QSS 风格）
- 按钮：隐藏到托盘 / 退出应用 / 取消

## 7. 当前 UI 状态结论

- 主界面视觉已达可用水平
- 核心截图路径可用
- 编辑工具与长截图入口均已接入，长截图仍按实验特性管理
