# UI 设计文档（同步版）

> 更新时间：2026-03-06

## 1. 设计目标

- 简约、现代、轻量
- 主界面非透明
- 截图工作流集中、反馈明确

## 2. 主界面结构

- 左侧导航栏
- 中央主内容区
- 右侧格式/常用设置区

## 3. 快速截图区（当前）

- 模式按钮：`全屏`、`区域`
- “滚动”不作为独立可用模式（已隐藏主路径）
- `开始截图` 触发当前模式流程

## 4. 覆盖层设计（SelectionOverlay）

- 半透明遮罩
- 选区有轻微蓝色描边（仅交互态）
- 选区尺寸信息浮层
- 工具条：画笔/矩形/圆形/马赛克/文字/长截图/保存/取消/确认

## 5. 长截图 UI 行为

- 入口：区域选区后的“长截图”按钮
- 滚轮期间选区可视高度持续增长
- 预览显示当前拼接结果
- 完成后可保存或复制

## 6. 当前 UI 状态结论

- 主界面视觉已达可用水平
- 核心截图路径可用
- 编辑工具属于“入口已存在、能力待实现”
