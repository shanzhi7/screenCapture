# UI 设计文档（按当前实现更新）

> 更新时间：2026-03-06

## 1 设计方向

- 风格：简约、轻盈、卡片化、弱对比渐变背景
- 交互：主界面聚合常用入口，截图时进入覆盖层工作流
- 技术：Qt Widgets + QSS + SVG 图标

---

## 2 主窗口结构

主窗口由三列/区组成：

1. 左侧导航栏
2. 中央主内容区
3. 右侧信息与设置区

### 2.1 左侧导航按钮

- `sideCaptureButton`：截图入口（占位）
- `sideGalleryButton`：图库/历史入口（占位）
- `sideConfigButton`：设置入口（占位）
- `sideBottomButton`：底部设置快捷入口（占位）

### 2.2 顶部操作按钮

- `btnPrev`：上一步视图（占位）
- `btnShotIcon`：截图视图入口（占位）
- `btnNext`：下一步视图（占位）
- `btnNewTask`：新建会话（占位）
- `btnHistory`：历史面板（占位）
- `btnTopSettings`：设置面板（占位）

### 2.3 快速截图卡片

- `modeFullButton`：全屏模式（已实现）
- `modeRegionButton`：区域模式（已实现）
- `modeScrollButton`：滚动模式（已实现最小可用版）
- `startCaptureButton`：启动截图（已实现）

### 2.4 最近截图卡片

- 网格容器：`recentGridLayout`
- 当前为演示数据卡片（占位数据）
- `btnMoreRecent`：更多入口（占位）

### 2.5 右侧卡片

- 格式卡片：`formatCard`，`btnMoreFormat`（占位）
- 常用设置卡片：
  - `btnHotkeySetting`（已接入触发截图）
  - `btnFormatSetting`（占位）
  - `btnAutoSave`（仅文案切换，逻辑待接入）

## 3 覆盖层（SelectionOverlay）

截图时显示全屏遮罩：

- 半透明暗层
- 可拖拽选区
- 选区尺寸提示
- 选区高亮

选区释放后显示工具条：

- 编辑按钮：画笔 / 矩形 / 圆形 / 马赛克 / 文字（待接入真实绘制）
- 操作按钮：保存 / 取消 / 确认（已接入基础流程）

---

## 4 滚动截图最小可用流程（T101）

1. 选择 `modeScrollButton` 并点击 `startCaptureButton`
2. 进入覆盖层框选滚动区域
3. 自动抓取首帧
4. 弹出控制面板：
   - 追加一帧
   - 完成拼接
   - 取消
5. 完成拼接后在右侧预览展示长图

---

## 5 提示系统

- 使用 `ShowTip` 浮动提示组件。
- 要求：所有未实现按钮必须有明确提示，避免无反馈。

---

## 6 样式策略（QSS）

- 全局字体统一
- 主卡片圆角 + 细边框
- 分段按钮选中态蓝色渐变 + 阴影
- 关键操作按钮橙色渐变
- 悬浮与选中状态有可感知反馈
